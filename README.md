# shareEat Project

A RN application developed for share Eat

## Setup

Clone this repo and go to project folder.

Run the scripts below in your terminal.

### iOS

* ```npm install```
* ```cd/ios``` and ```pod install```
* ```react-native run-ios```

### Android

* ```npm install```
* ```react-native run-android```
