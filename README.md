# shareEat Project

A RN application developed for share Eat

## Setup

Run the scripts in your terminal at the project folder.

### iOS

* ```npm install```
* ```cd/ios``` and ```pod install```
* ```react-native run-ios```

### Android

* ```npm install```
* ```react-native run-android```
